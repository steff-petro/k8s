# k8s

A duide to install a simple cluster

## Task List

- [ ] Setup infrastructure using: https://github.com/steff-petro/tf-playground
- [ ] Configure instances to be ready for configuration using: https://github.com/steff-petro/ansible-playground
- [ ] Run sudo kubeadm init on control node and follow instructions from the output that need to be completed on the worker nodes
- [ ] Follow helm installation instructions: https://helm.sh/docs/intro/install/
- [ ] Follow cilium installation instructions: https://docs.cilium.io/en/stable/installation/k8s-install-helm/
